## 核心结论预览

本报告不建议把南京泛成岗位理解成“宠物食品品牌方岗位”或“轻松出国岗位”。公开信息更支持的画像是：泛成属于 B2B 食品配料、营养原料、精细化工、动物营养与宠物食品原料供应链公司；岗位更接近海外客户开发、询盘报价、样品/技术资料协调、订单推进、回款和复购维护的国际业务/外贸销售。

最终建议是有条件判断：只有在候选人能够接受销售 KPI、收入波动、假期减少、客户开发压力，并在入职前拿到明确薪酬结构、试用期 KPI、提成兑现规则、客户资源分配、出差机制和离职止损方案时，南京路径才值得作为“离开教师路径、换取国际商业经验”的高波动选择。若候选人的首要目标是稳定、长假和低压，学校编制岗位仍是更稳的基线。
