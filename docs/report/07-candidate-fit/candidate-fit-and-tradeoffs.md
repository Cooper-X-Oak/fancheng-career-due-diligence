# 候选人适配与取舍

## 本章调研问题

候选人的显性需求和隐性需求是什么；南京泛成路径和学校编制路径分别匹配哪些需求、冲突哪些需求。

## 候选人的需求结构

显性需求：

- 稳定。
- 较长年假。
- 较高收入。
- 出国机会。
- 国际业务。
- 经济独立。

隐性需求：

- 离开被安排的教师路径。
- 获得自决权。
- 与家庭控制保持距离。
- 证明英语能力可以变成商业能力。
- 获得未来外企或更好商业岗位的筹码。

这些需求之间存在冲突。稳定和长假更接近学校编制；较高收入上限、国际业务、自决权和商业经验更接近南京路径；低压与销售 KPI 冲突；出国机会与新人阶段现实冲突。

## 南京路径适配

南京路径匹配：

- 国际业务兴趣。
- 英语沟通能力。
- 希望离开教师身份。
- 希望经济独立和地理距离。
- 希望未来进入外企或更市场化岗位。

南京路径冲突：

- 抗压能力一般。
- 未验证销售、KPI、客户开发和收入波动适配度。
- 想要稳定和长假。
- 对提成兑现和客户资源不确定。
- 对海外差旅可能存在理想化预期。

适配结论：南京路径不是舒适选择，而是职业自决和商业试错选择。它适合愿意用 6-12 个月承受压力来验证商业适配的人，不适合把“出国、英语、宠物食品”理解成轻松标签的人。

## 学校路径适配

学校路径匹配：

- 稳定。
- 长假。
- 社会认可。
- 家庭可接受。
- 风险低。

学校路径冲突：

- 不期待长期当老师。
- 可能延续家庭控制。
- 国际业务和商业能力沉淀不足。
- 未来转外企或 B2B 商业岗位更难。
- 经济独立和地理距离可能不足。

适配结论：学校路径适合把稳定和生活秩序放在第一位的人；如果候选人对教师身份的排斥是长期、强烈和不可调和的，学校路径会变成低风险但高后悔的选择。

## 不可兼得的取舍

| 需求 | 学校编制 | 南京泛成 |
| --- | --- | --- |
| 稳定 | 强 | 中弱，取决于业绩和团队 |
| 长假 | 强 | 弱，公开信息不足 |
| 高收入上限 | 中弱 | 中高，但依赖销售结果 |
| 出国机会 | 弱 | 中，但新人不确定 |
| 国际业务 | 弱 | 强于学校 |
| 经济独立 | 中 | 中高，取决于收入和城市成本 |
| 自决权 | 取决于家庭距离 | 较强，但伴随压力 |
| 低压 | 中 | 弱 |

## 心理风险

候选人抗压能力一般，且商业销售适配未验证。南京路径的心理风险不是抽象的“辛苦”，而是：

- 连续开发无回复。
- KPI 临近但订单未落地。
- 客户反复比价。
- 内部协调慢但客户催促。
- 提成发放时间长。
- 对出国期待落空。
- 与家庭因放弃编制产生冲突。

如果缺少支持系统和止损计划，南京路径可能迅速变成自我怀疑。

## 适配总结

候选人不是“明显应该去南京”，也不是“明显应该留学校”。真正的判断是：

- 如果候选人最怕的是人生被教师路径固定，南京路径更符合长期自决。
- 如果候选人最怕的是不稳定、销售压力和收入波动，学校路径更符合短期安全。
- 如果南京 Offer 不能明确薪酬、KPI、客户资源、差旅和培训，放弃编制风险过高。
- 如果学校岗位让候选人强烈排斥且看不到转向商业的路径，留在学校会形成长期后悔。

## 本章信息缺口

- 候选人真实抗压阈值、家庭压力强度、现金储备和可接受试错时间公开信息未能确认。由于该信息对决策重要，因此构成信息不透明风险。
- 南京岗位的实际上级、团队氛围和绩效宽容度公开信息未能确认。由于该信息对决策重要，因此构成信息不透明风险。
- 学校岗位的具体工作强度和家庭距离公开信息未能确认。由于该信息对决策重要，因此构成信息不透明风险。
