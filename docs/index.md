# 择业调研与雇主背调

本项目用于承载“南京泛成外贸销售岗位 vs 学校编制岗位”的择业调研、雇主背调、行业上下游分析、社交平台口碑调研、职业路径推演和最终决策建议。

## 阅读入口

- [报告索引](./report/)
- [交付规范](./report/delivery-spec)
- [一次性调研任务 Prompt](./goal-prompt)

## 项目原则

- 所有公司、行业、岗位、口碑和职业路径判断都通过公开网页搜索与 agent-reach 能力完成。
- 每个细分点单独形成 Markdown 文件，单独检索，单独交叉循证。
- 不设置任何“请填写”内容；公开信息无法确认时，直接写为信息不透明风险。
- 最终建议必须区分事实、推断、风险和建议。
